How to convert a .safetensors checkpoint for browser use
  1. Install Python 3.10+ and pip install diffusers optimum onnxruntime accelerate safetensors.
  2. Run python -m optimum.exporters.diffusers.onnx --model PATH_TO_SAFETENSORS --task stable-diffusion-xl --opset 17 --device cpu --output exported-model.
  3. Zip the contents of exported-model and load the archive above.
